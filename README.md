# Data Cleaning and Visualization Project

## Project Overview

This project demonstrates the process of cleaning a raw dataset and visualizing meaningful insights using Python. It includes data preprocessing techniques such as handling missing values, removing duplicate records, detecting and removing outliers, and creating various visualizations.

The project was developed as part of the **Thiranex Internship**.

---

## Objectives

- Clean a raw dataset using Python.
- Handle missing values.
- Remove duplicate records.
- Detect and remove outliers.
- Visualize important insights using charts.
- Save the cleaned dataset for future analysis.

## Features

- Dataset loading
- Missing value handling
- Duplicate record removal
- Outlier detection using IQR
- Data preprocessing
- Data visualization
- Save cleaned dataset
- Generate multiple charts

---

## Data Cleaning Process

1. Load the dataset.
2. Check for missing values.
3. Fill missing numerical values with the mean.
4. Fill missing categorical values with the mode.
5. Remove duplicate records.
6. Detect and remove outliers using the IQR method.
7. Save the cleaned dataset.
